02 · CPG 精确落脚控制

腿式机器人精确落脚与分层控制框架

在 UC Berkeley Hybrid Robotics Lab 参与通用精确落脚控制研究，目标是在复杂地形与多障碍场景中建立更通用的腿式机器人运动控制框架。

  • 基于 Bezier-Augmented CPG 生成参考步态，低层控制策略 50Hz，对接 1kHz 关节级 PD 控制。
  • 高层 Planner 10Hz 在高度图上规划落脚点，视觉编码采用 3-layer CNN + 3-layer MLP，训练算法为 PPO。
  • 任务覆盖踏石跨越（含仿真踏石策略验证）、障碍规避与初步 sim2real 验证，感知链路使用 RealSense T265 与 D435。
CPG Residual Bezier PPO Height Map Sim2Real
03 · 强化学习精准踢球

腿式机器人地形自适应精准踢球

围绕未知地形下的精准射门问题，构建稳健运动控制、踢球规划和多摄像头感知的完整技术链路。

  • 以连续三次射门的状态-动作-转移历史为输入，尝试在 2-3 次试射内适应陌生地形动态。
  • 使用 Transformer 进行离线序列学习，并通过 goal relabeling 与环境随机化扩展训练数据。
  • 通信侧使用 UDP 与 Ros_Bridge 打通 Python3、ROS 与感知执行链路，多摄像头负责足球检测与导航辅助。
Transformer Offline RL Goal Relabeling UDP / ROS Multi-camera
04 · 移动机器人导航

轮式机器人路径规划与动态避障

在西安交通大学机器人与智能系统研究所担任队长，围绕动态环境中的导航效率和避障稳定性完成整套系统设计。

  • 基于激光雷达进行动态障碍物规避，避障成功率提升 35%。
  • 结合 Relaxed A* 与 TEB 进行全局与局部规划，导航效率提升 25%。
  • 同步完成 SolidWorks 建模、ANSYS 应力分析与结构优化。
Lidar Relaxed A* TEB Dynamic Avoidance
